RIT University News

A new university news website engaging for students and faculty. This project was a visual design focus on improving typographical elements, improving content enrichments and improve the RIT University webpage.

Client: School Project

Role: Visual Design

Duration: 3 weeks

Problem

Visual hierarchy

The website lacks a visual hierarchy for the user to scan. The left rails and the right rails have an uneven amount of content which makes the user confused while looking for something to read.



Viewing patterns

The left body and the right rails are unbalanced. I wanted to improve the layout of this website so it helps give the user a better scanning pattern.


How can we create a fun reading environment that gets students and faculty up to date with ongoing events at RIT?

Goals



Live feedback

How can we get live feedback from the users' reactions?

Rich Contents

How can we get the student to continue visiting the website?

Organized

How can we make the website easier to read the contents?

Design

A cleaner, clearer layout with better alignments and content enrichment will keep students and faculty more engaged in reading the RIT news website.

Live Feedbacks

I wanted the user to be engaged in the articles they read. It gives an idea of what articles are best to read.

Tyopgraphical hierarchy

Tyopgraphical hierarchy gives the user a better way to scan the website. I wanted the header and the brief bigger. This helps the user find their reading interests quicker.

The Pause

I intentionally put a darker shade on the web content to direct the user to videos. This will make users pause in reading, view the view and return to reading articles. A break from reading.

Live Feedbacks

I wanted the users to be involved with the articles to give feedback about what articles are most interesting.

Better grouping

I have broken down the information on the website into sections. I wanted the content to go by different events, colleges, etc. I have them intermixed on top of the website for the top story and then other topics.

Conclusion

Overall, this was an enjoyable visual design project that helps me focus on iconography, typography, visual hierarchy, and much more. I like this project because it gave me a better sense of building a website visually and also explain my thoughts with reasons.

Finish, Thanks!
