DMV MV-83 Form Redesign

The DMV MV-83 is a form that users fill out to register a new car. The form has a lot of redundancy and information for users to fill out; which can overwhelm the applicants. I chose to work on this project because I want to help reduce the frustration of many users filling out this DMV form.

Client: School Project

Role: Interaction Design

Duration: 3 weeks

Problem



The DMV form has a lot of redundancy.


The form is cluttered which can feel daunting.


The DMV needs a better way to engage the user to help them not feel bothered.


How can we create an interface that users can fill out forms in an easier way?

Goals



Reduction

Reducing the information load to help user breakdown the information processing.

Engaging Visuals

Make the form more visually enhancing for the users.

Faster Filling

Improving the information layout to help the user fill out the from faster.

Solutions



Mobile View

Breakdown to smaller steps while the user can see how much they need to fill in a way to help reduce the overwhelming feeling.






Accordion-style guides users to breakdown into simple steps to complete the process.







Desktop View

With consideration, I wanted the mobile view and desktop view to have the same elements. So if the user is doing this on the desktop would easily scale down to mobile. I made sure my design can serve both viewports.





Conclusion

This was a fun project creating a form as an opportunity to improve the user interaction by creating an app that can save a lot of frustrations. I would improve one part of the form for more specific users who use commercial cars or special cars. I have made the form part there really long and confusing. I would improve that by asking the user their profile so not all the questions are needed to be upfront for them.
